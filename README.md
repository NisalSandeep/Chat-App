## в—Ҳ Running Locally

**1 вҖ” Clone the repo**
```bash

```

**2 вҖ” Start the backend**
```bash
cd backend
npm install
npm run dev
```

**3 вҖ” Start the frontend**
```bash
cd frontend
npm install
npm run dev
```

**4 вҖ” Open your browser**
```
http://localhost:5173
```
